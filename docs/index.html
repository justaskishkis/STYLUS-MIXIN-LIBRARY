Introduction:
as we write stylus(or any preprocessor) code we want to save time, make our code more robust so we use "mixins".
Mixins documented below are a copy of most popular SCSS mixin library https://www.bourbon.io/docs/5.1.0/, with some tweaks and some features removed.
In alphabetical order:

.some-class
 border-color_(red #fff blue yellow)

border-top-color red
border-right-color #fff
border-bottom-color blue
border-left-color yellow

prefixer()

.some-class
 prefixer_(opacity, 0.2, o moz)

NOTE:

"o moz" is OPTIONAL parameter,
default value is "webkit moz ms o", stored in _settings.styl
variable "$vendorPrefixList"

Result:

-o-opacity 0.2
-moz-opacity 0.2
opacity 0.2

size()

.some-class
 size_(1px, 2px)

NOTE:

both values must be entered, if you just need 1 parameter, enter simply:
width 1px
OR
height 2px

Result:

width 1px
height 2px

triangle()

.some-class
 triangle_('up', 20px, 10px, red)

NOTE:

all possible options for direction are:
'up', 'up-right', 'right', 'down-right', 'down', 'down-left', 'left', 'up-left'

Result:

display block

border-bottom-style solid
border-left-style solid

border-bottom-color #f00
border-left-color transparent
border-top-width 0

border-left-width 10px
